# Split It

**Created:** 2026-04-02
**Owner:** Soubhagya Bhunia

## Short Description

Split It is a cross-platform mobile app to track, share and settle expenses within groups. It makes it easy for friends, roommates and travel companions to record shared costs, compute each member's liability, and settle balances quickly.

## Goals

- Provide a simple, delightful way to add expenses and split them among group members.
- Show clear per-group balances and per-person liabilities.
- Make settling easy (links, UPI/payments, or manual settle-offs).
- Keep user data private and secure by using row-level security and minimal personal data.

## Non-goals (MVP)

- Immediate support for advanced bank reconciliations or payment processing beyond UPI links and shareable settlement flows.

## Key Stakeholders

- Soubhagya Bhunia (owner)
- Early test users: friends, roommates, small travel groups

## Success Metrics

- Time-to-add-expense under 10s for new users
- 80% of test users can create a group and add an expense without help
- Daily active users and retention for group creators
